# Youtube-Playlist-Sync
 Script to download all videos from a youtube playlist, convert videos to a mp3 file and then upload it to s3 storage
 
- To start create a credentials.py file to store:
'''
Youtube_API_Key = ''
Playlist_ID = ''

AWS_Key = ''
AWS_Secret_Key = ''
AWS_Region = ''
AWS_Bucket = ''
'''
